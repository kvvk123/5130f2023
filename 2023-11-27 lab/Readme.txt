This project is a Progressive Web App (PWA) featuring user authentication capabilities. It allows users to sign-up, sign-in, and access as guests.

Primary Files:
- server.js: Node.js server setup with Express.
- service-worker.js: Service worker for PWA offline capabilities.
- HTML Files: signin.html, signup.html, guest.html, index.html, success.html, and offline.html.
- script.js: Front-end JavaScript for user interaction handling.
- db.js: Database configuration for user management.
- manifest.json: PWA manifest detailing app properties.
