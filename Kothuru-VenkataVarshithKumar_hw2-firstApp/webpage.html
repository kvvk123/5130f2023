Attributes Used in HTML and CSS Files

HTML and CSS attributes are utilized for content styling and structure:

  • class: Used to apply specific styles or behavior to elements with the same class.
  • id: Provides a unique identifier for an HTML element.
  • href: Specifies the URL for hyperlinks.
  • table: Represents tabular data in rows and columns.
  • tr: Defines a row in a table.
  • th: Represents a header cell in a table.
  • td: Represents a data cell in a table.
  • ol: Creates an ordered list, which is a list of items with a defined sequence.
  • ul: Creates an unordered list, which is a list of items without a specific order.
  • li: Represents a list item within an ordered or unordered list.
  • p: Defines a paragraph of text, creating a block-level element for organizing content.
  • src: Defines the source (URL) for images.
  • alt: Provides alternative text for images (useful for accessibility).
  • target: Specifies how links should open (e.g., in a new tab or in the same window).
  • border: Sets the border properties for elements.
  • text-align: Aligns the text content within an element.
  • background: Sets the background color or image for elements.
  • color: Sets the font color for elements.
  • padding: Adds space inside an element's boundaries.
  • margin: Adds space outside an element's boundaries.
  • display: Defines the type of box an element generates.
  • float: Specifies how an element should float.
  • position: Specifies the type of positioning method used for an element.
  • width: Sets the width of an element.
  • height: Sets the height of an element.
  • max-width: Sets the maximum width of an element.
  • opacity: Sets the transparency of an element.
